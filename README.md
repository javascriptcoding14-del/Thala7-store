# Thala7-store

## What is this project?

This is a small online shop website made with HTML and CSS. It has pages for:

- Home page (`index.html`)
- Product details page (`product.html`)
- Shopping cart page (`shopping_cart.html`)
- Payment page (`payment-section.html`)

The style files are in the `css/` folder and the images are in the `images/` folder.

## What does it do?

- Shows product information
- Lets users view a cart page
- Shows payment details layout
- Has a clean shop design for a simple store website

## Files in the project

- `index.html` - store home page
- `product.html` - product detail page
- `shopping_cart.html` - cart page
- `payment-section.html` - payment page
- `css/style.css` - main styles
- `css/product.css` - product page styles
- `css/shopping_cart.css` - shopping cart page styles
- `css/payment-section.css` - payment page styles
- `images/` - images used in the website

## How to run

1. Open the project folder in your file manager.
2. Double-click `index.html` to open the website in your browser.

Or:

1. Open the folder in a code editor like VS Code.
2. Right-click `index.html` and choose `Open with Live Server` if you have the Live Server extension.

## Notes

- This is a static website, so no server or backend is needed.
- Just open the HTML files in a browser to see the pages.
